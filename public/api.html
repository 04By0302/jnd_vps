加拿大28开奖API接口文档

本文档详细介绍了PC28加拿大开奖数据API的所有接口，包括实时开奖查询、遗漏统计、已开统计、AI智能预测等功能的使用方法。

返回首页 历史数据下载

基础信息

API基础地址： https://pc28.help
数据格式： JSON
请求方式： GET
字符编码： UTF-8

使用说明： 所有接口完全免费开放，请控制请求频率不小于1秒/次，避免对服务器造成过大压力。

1. 实时开奖数据API

接口地址

GET /kj.json

请求参数

参数名 类型 必填 说明
limit number 否 返回记录数量，默认1，最大100

调用示例

// 获取最新1期开奖数据
https://pc28.help/kj.json?limit=1

// 获取最近10期开奖数据
https://pc28.help/kj.json?limit=10

返回示例

{
  "message": "success",
  "data": [
    {
      "qihao": "3392278",
      "opentime": "2026-02-03 18:58:00",
      "opennum": "6+9+2",
      "sum": "17"
    }
  ]
}

返回字段说明

字段名 类型 说明
qihao string 期号
opentime string 开奖时间（格式：YYYY-MM-DD HH:mm:ss）
opennum string 开奖号码（格式：a+b+c）
sum string 和值（三个号码之和，范围0-27）

2. 遗漏统计API

接口地址

GET /yl.json

功能说明

返回当前遗漏统计数据，包括各统计类型（大小、单双、组合、和值等）距离上次出现的遗漏期数。

调用示例

https://pc28.help/yl.json

统计类型说明

类型代码 说明 值
da / xiao 大小 大（和值14-27） / 小（和值0-13）
dan / shuang 单双 单（奇数） / 双（偶数）
dd / xd / ds / xs 组合 大单 / 小单 / 大双 / 小双
00-27 和值 28种和值统计
jd / jx 极值 极大（22-27） / 极小（0-5）
bz / dz / sz / zl 形态 豹子 / 对子 / 顺子 / 杂六
xb / zhong / db / bian 边中 小边（0-9） / 中（10-17） / 大边（18-27） / 边（小边+大边）
long / hu / he 龙虎合 龙（首位>末位） / 虎（首位<末位） / 合（首位=末位）

3. 已开统计API

接口地址

GET /yk.json

功能说明

返回今日已开统计数据，统计今天各类型（大小、单双、组合、和值等）的出现次数。数据在每日0点自动重置。

调用示例

https://pc28.help/yk.json

说明： 统计类型与遗漏统计相同，返回的是各类型在今日已开奖期数中的出现次数。

4. AI智能预测API

接口列表

接口地址 说明
/ds.json?limit=10 单双预测
/dx.json?limit=10 大小预测
/zh.json?limit=10 组合预测（大单/小单/大双/小双）
/sha.json?limit=10 杀组合预测（建议避免的组合）

请求参数

参数名 类型 必填 说明
limit number 否 返回记录数量，默认10，最大100

注意： AI预测结果仅供参考，预测准确率实时显示在首页。预测数据由AI模型分析历史数据生成，不保证100%准确。

5. 胜率统计API

单双胜率

GET /api/winrate/danshuang

大小胜率

GET /api/winrate/daxiao

组合胜率

GET /api/winrate/combination

杀组合胜率

GET /api/winrate/kill

返回示例

{
  "message": "success",
  "winRate": "68.5",
  "hits": 68,
  "total": 100
}

6. 数据导出API

开奖数据导出

GET /api/export/lottery?limit=1000

导出Excel格式的历史开奖记录，最多支持30000期。

统计数据导出

GET /api/export/stats?days=30

导出Excel格式的统计数据，最多支持90天。

使用建议

  • 建议使用HTTPS协议访问，确保数据传输安全
  • 请求频率建议控制在1秒/次以上，避免频繁请求
  • 建议在客户端做好缓存处理，减少不必要的请求
  • 开奖数据实时更新，建议在开奖后2-5秒后查询
  • AI预测数据需要计算时间，可能略有延迟
  • 大量数据导出建议使用Excel导出接口

错误码说明

错误码 说明
200 请求成功
400 请求参数错误
429 请求过于频繁
500 服务器内部错误
503 服务暂时不可用
